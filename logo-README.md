# FigureYa Logo

## Design Elements

The FigureYa hexagonal logo incorporates:

- **Hexagon shape**: Represents structure, precision, and the modular nature of the framework
- **Blue gradient background**: Symbolizes science, trust, and biomedical research
- **Data visualization elements**:
  - Bar chart (left): Represents quantitative analysis
  - Line chart (right): Shows trends and temporal data
  - DNA helix accent (top): Emphasizes biomedical/genomic focus
- **Typography**: 
  - "Figure" in white for clarity
  - "Ya" in yellow (#F4E04D) for emphasis and warmth

## Color Palette

- Primary blue: `#2C5F7C` (hexagon border)
- Secondary blue: `#3A7CA5` (inner hexagon)
- Accent cyan: `#81C3D7` (bar charts)
- Accent yellow: `#F4E04D` (line chart & "Ya" text)
- Accent coral: `#E8A87C` (DNA helix)

## Files

- `logo.svg` - Vector format (scalable, recommended for web and print)
- `logo.png` - Raster format (for quick use)

## Usage

```markdown

```
